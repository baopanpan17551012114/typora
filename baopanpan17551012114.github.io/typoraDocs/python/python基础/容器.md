str, list, tuple, set, dict

|              容器               |          特性          |       访问       |
| :-----------------------------: | :--------------------: | :--------------: |
|            str-'abc'            |  有序、可重复、不可变  |   不可下标访问   |
|         list-[1, 2, 3]          |   有序、可重复、可变   | 可下标访问、切片 |
|       set-{1, 2, 3, '4'}        | 有序、不可重复、不可变 |   不可下标访问   |
| tuple-(12, 34.56, 'abc', 'xyz') |  有序、可重复、不可变  | 可下标访问、切片 |
|      dict-{'a': 1, 'b': 2}      |  无序、不可重复，可变  |                  |
